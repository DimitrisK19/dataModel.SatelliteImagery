Description globale : **Cette entité contient une description harmonisée d'une couche de données générique EOGeoDataLayer conçue pour le domaine de l'imagerie satellitaire. Cette entité est principalement associée aux couches de données de sortie liées aux applications d'analyse de l'observation de la Terre.  
version : 0.0.1
