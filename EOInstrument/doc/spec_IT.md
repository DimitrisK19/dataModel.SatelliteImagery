Descrizione globale: **Questa entità contiene una descrizione armonizzata di un generico EOInstrument fatto per il dominio Satellite Imagerry. Questa entità è principalmente associata agli strumenti satellitari relativi alle applicazioni di Analisi dell'Osservazione della Terra.**  

## Elenco delle proprietà  

- `alternateName`: Un nome alternativo per questa voce  - `carriedOn`: L'ID della piattaforma satellitare su cui è trasportato lo strumento  - `dataProvider`: Una sequenza di caratteri che identifica il fornitore dell'entità di dati armonizzata.  - `dateCreated`: Timestamp di creazione dell'entità. Questo sarà di solito assegnato dalla piattaforma di archiviazione.  - `dateModified`: Timestamp dell'ultima modifica dell'entità. Questo sarà di solito assegnato dalla piattaforma di archiviazione.  - `description`: Una descrizione di questo articolo  - `id`: Identificatore unico dell'entità  - `instrumentID`: Una stringa di testo obbligatoria usata per dichiarare l'ID del carico utile dello strumento  - `instrumentName`: Una stringa di testo obbligatoria usata per dichiarare il nome del carico utile dello strumento  - `name`: Il nome di questo articolo.  - `operationalMode`: Una stringa di testo usata per dichiarare le modalità del sensore supportate, se disponibili  - `owner`: Una lista contenente una sequenza di caratteri codificata in JSON che si riferisce agli ID unici dei proprietari  - `polarizationMode`: Una stringa di testo usata per dichiarare le modalità di polarizzazione, se disponibili  - `seeAlso`: elenco di uri che puntano a risorse aggiuntive sull'elemento  - `source`: Una sequenza di caratteri che dà la fonte originale dei dati dell'entità come URL. Si raccomanda di essere il nome di dominio completamente qualificato del fornitore di origine, o l'URL dell'oggetto di origine.  - `swathID`: Una stringa di testo usata per dichiarare l'ID dell'andana, se disponibile  - `type`: Tipo di entità NGSI-LD. Deve essere uguale a EOInstrument.
